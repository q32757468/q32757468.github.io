十一、在实际使用过程中遇到的一些问题以及解决办法

在实际的使用过程中使用markdown在hexo编写文章有时会遇到一些意想不到的问题，这里来进行一个汇总以及解决办法：

  • 在hexo中使用表格的时候在vscode中预览的时候是好好的，可是当打开网页之后却会出现表格上方会出现大量的留白的情况，检查网页代码发现是网页中多了很多的换行符

    那么解决办法是在表格使用 `` 与 ``包裹起来，然后再回去查看网页，ok，问题解决。
